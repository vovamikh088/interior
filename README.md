# Интерьер — интернет-магазин мебели 🛋️

Это многостраничный адаптивный сайт для магазина со qвременной мебели. Проект выполнен с использованием методологии БЭМ и препроцессора SCSS.

[**Посмотреть проект вживую**](https://vovamikh088.github.io/interior/)

<img src="img/preview.png" width="400" alt="Превью сайта Интерьер">

## Технологии в проекте
- **HTML5** (семантическая верстка)
- **CSS3 / SCSS** (переменные, миксины, вложенность)
- **БЭМ** (структурирование стилей и классов)
- **Flexbox & Grid** (построение сеток)
- **Адаптивный дизайн** (Desktop, Tablet, Mobile)

## Особенности
- **Многостраничность:** Главная страница, Каталог товаров и Корзина.
- **Интерактив:** Настроенные формы поиска, переходы по кнопкам и адаптивное меню.
- **Оптимизация:** Быстрая загрузка за счет чистого кода и отсутствия тяжелых библиотек.

## Структура файлов в корне
- `index.html`, `catalog.html`, `cart.html` — страницы сайта
- `style.scss` — основной файл стилей
- `_vars.scss` — переменные проекта (цвета, шрифты)
- `_cart.scss` — стили для страницы корзины
- `style.css` — итоговый скомпилированный файл
- `img/` — папка с графикой

## Как запустить локально
Для корректного отображения всех путей рекомендуется запустить локальный сервер:

```bash
# С помощью Node.js (если установлен)
npx serve

# С помощью Python
python -m http.server 8000
```
Или просто откройте файл index.html в вашем браузере.
